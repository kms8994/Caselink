# 평가 정책

## 검색 품질

- 관련 조문이 입력된 경우 같은 참조조문을 가진 판례가 먼저 나와야 한다.
- 자연어 입력은 관련 가능성이 있는 조문과 사실관계 키워드를 함께 사용한다.
- 내부 점수는 정렬에만 사용하고 사용자 화면에는 퍼센트로 노출하지 않는다.

## 비교 품질

- 비교 설명은 저장된 구조화 데이터 안에서만 생성한다.
- 결론 라벨이 달라도 사실관계가 동일하다고 단정하지 않는다.
- 모든 비교 화면에 공식 원문 확인 권고를 표시한다.

## 피드백

- 도움 됨, 관련성, 결론 라벨 오류 신고를 MVP 피드백으로 수집한다.
- 오류 신고가 누적된 판례는 `needs_review` 대상으로 재검수한다.
